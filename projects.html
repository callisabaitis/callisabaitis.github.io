× Hearts Card Game
Language: OCaml
Completed: Fall 2019
Class: CS 3110: Data Structures and Functional Programming
Description:
We created the card game Hearts with OCaml. The user can play the game with up to 4 people and choose the automated player(s) to be on easy mode (chooses cards randomly) or hard mode (chooses cards intelligently). The hard mode strategically chooses a card to play based on factors such as the cards previously played, what turn it is in the round, and if any cards that are worth points have been played. We created a module form.ml that creates an interface that displays each players’ score, the current cards played, and the current player’s hand if a player’s hand should be shown.
